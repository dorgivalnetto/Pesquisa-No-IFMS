# pesquisaNoIFMS
O conteúdo dessa página apresenta um resumo dos PROCEDIMENTOS PARA SOLICITAÇÃO DE AUTORIZAÇÃO DE PESQUISA NO ÂMBITO DO IFMS. A página foi criada pelo Coordenador de Pesquisa e Inovação (COPEI) do Campus Corumbá, professor Me. Dorgival Netto (Portaria IFMS nº 1.485 de 12 de Junho de 2017), a fim de facilitar a apresentação das informações para os estudantes dos cursos de Pós-Graduação do Campus Corumbá.

Destacamos que os procedimentos Oficiais são encontrados por meio dos canais Oficiais do IFMS.
